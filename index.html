Git/Github

Publisert 21.09.2018.

Git er programvare som brukes i forbindelse med versjonskontroll. GitHub er en nettbasert hosting service som tilbyr tjenester med Git-teknologi.

Media Typer

Publisert 27.09.2018.

Det er i all hovedsak kun 2 mediatyper som brukes i dag, media="screen" og media="print". Mediatypen "screen" brukes til å definere at et stilark er ment for en skjerm, "print" brukes for å lage en printervennlig version av siden.

Responsivt design

Publisert 05.10.2018.

Med responsivt design, snakker vi i denne sammenheng om at nettstedene vi lager skal respondere på skjermstørrelsen til klient-enheten. Det er flere måter å oppnå dette på, som media query, media type, responsive størrelser (for eksempel %, vw, vh, osv).
