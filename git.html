Git / Github

Publisert 21.09.2018.

Git

Git er programvare som brukes i forbindelse med versjonskontroll. Man har et repository, en type mappe som oppdateres når man gjør endringer i en lokal mappe. Så fort Git ser at den lokale mappen ikke inneholder det samme som i repository, får man muligheten til å synkronisere. Man kan velge å overskrive gammel versjon med den nye.

GitHub

GitHub er en nettbasert hosting service som tilbyr tjenester med Git-teknologi. Denne tjenesten gjør det også mulig å samarbeide på én kode, samtidig, uten at man overskriver hverandre. Dette sørger versionskontrollen for.

GitHub tilbyr både en betalingstjeneste og en gratis. Forskjellen er at "private repositories" kun er tigjengelig mot betaling, samtidig som "public repositories" er for alle. I Juni 2018 hadde GitHub over 28 millioner brukere med til sammen 57 millioner repositories (inkludert 28 millioner public repositories).

Det er lett å skjønne hvorfor GitHubs nettjeneste har såpass mange brukere, når teknologien fungerer såpass smertefritt. GitHub er faktisk så vellykket at Microsoft har inngått en kjøpsavtale verdt 7,5 milliarder amerikanske dollar!
